# AIBL Network (aibl-network)

This repository hosts the official web application for AIBL Network.

## Overview
AIBL Network is a decentralized protocol for immutable AI memory and cognitive history.
이 프로젝트는 AIBL Network를 위한 웹 애플리케이션 저장소입니다.

## Deployment Info
- **Production URL:** https://aibl.network
- **Platform:** Vercel (Managed by AIBL Core Team)
